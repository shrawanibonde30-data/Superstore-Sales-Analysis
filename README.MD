# Superstore Sales Analysis

## Project Overview

This project analyzes 9,994 retail sales transactions using Python and MySQL. The dataset was cleaned using Pandas and stored in a relational MySQL database for business analysis.

---

## Tools & Technologies

- Python
- Pandas
- MySQL
- SQL
- Git & GitHub

---

## Project Workflow

1. Cleaned the dataset using Pandas
2. Performed date conversion and handled missing values
3. Designed the database schema in MySQL
4. Inserted 9,994 records into MySQL
5. Performed SQL-based sales and profitability analysis
6. Automated SQL execution using Python

---

## Key Business Insights

- The Technology category generated the highest revenue.
- Higher discounts significantly reduced profit margins.
- The West region contributed the highest total sales.
- Identified the top 10 most profitable products.
